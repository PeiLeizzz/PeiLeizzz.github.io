# 基本数据类型及注意点

  1. int(long int) 类型，取值范围：，记忆：大约范围 ，最多十位，占 字节
  2. long long 类型，取值范围：，记忆：大约范围 ，占 字节
  3. float 类型，取值范围：，实际精度： 位，一般很少用
  4. double 类型，取值范围：，实际精度： 位
  5. char 类型，取值范围：，0、A、a 的 ASCII 码值分别为 48，65，97，用单引号(' ')包裹

关于数据类型一些注意点：

  • 模运算的除数不能为 0
  • 一般程序中 define INF 或 const int INF 设置无穷大常数一般值为 ，更常见的为 ，因为int类型的上限为 ，设置为 的话两个INF相加不会溢出。具体语句为 define INF 0x3fffffff 或 const int INF = (1 << 30) - 1

# scanf、printf 以及常用 math 库函数

头文件：cstdio cmath

要点：

  • scanf 进行输入时，变量前必须有取址符&
类型 scanf printf
int %d %d
long long %lld %lld
float %f %f
double %lf %f
char %c %c
char[] %s %s

此表中需要注意的是：double 的输出也是用的%f

  • 控制输出格式
    • 整数控制位数：%nd，例如 printf("%5d\n", a); 位数不足高位用空格补齐。
    • 整数控制位数( 0 补齐)：%0nd，例如 printf("%05d\n", a); 位数不足高位用 0 补齐。
    • 小数控制位数：%.mf，例如 printf("%.5f\n", d); 此处为四舍六入五成双，四舍五入可以使用 round() 函数。
  • 使用 getchar和 putchar 输入/输出字符
    • getchar()只能吸收单个字符，包括换行符！所以使用时一定记得留意换行符！
    • putchar()输出单个字符
  • 常用math库函数（返回值都是 double 类型）
    • double fabs(double x): 返回 double 类型的参数的绝对值
    • double floor(double x) 和 double ceil(double x): 第一个是向下取整，第二个是向上取整，返回值都是 double 类型
    • double pow(double r, double p) : 返回 的值，double 类型
    • double sqrt(double x) : 返回 double 类型变量的算术平方根
    • double log(double x) : 返回 double 类型变量的以自然对数为底的对数值，如果要以任意底数为对数，则需要使用换底公式
    • double sin(double x) 、double cos(double x) 、double tan(double x)
    • double asin(double x) 、double acos(double x) 、double atan(double x)
    • double round(double x) : 返回参数四舍五入后的值

# 数组注意点

  • 数组大小必须是整数常量，不可以是变量
  • 局部定义的数组初始化时若只给部分元素赋初值，那其他未赋值元素默认值为 0 ；如果所有元素都未初始化，那么所有元素的值都是随机数
  • 全局定义的数组如果未初始化，那么所有元素的值默认为 0
  • 大数组（ ）需要定义在主函数之外（全局变量），避免爆栈
  • 数组元素赋相同值：
    • memset(arr, val, sizeof(arr)) 注意：是按字节赋值，所以一般赋 0 或 -1 值时使用
    • fill(arr, arr + n, val) 不是按字节赋值

# 字符数组

  • 初始化：char str[2] = {'a', '\0'}; 以结束符 '\0' 结尾，所以开字符数组一定要比实际长度至少多 1
  • 字符数组也可以用直接赋值的方法初始化：char str[3] = "ab"; 但仅限初始化，其它情况不允许直接赋值
  • 字符数组的输入输出：

    • scanf 和 printf

      • char str[10]; scanf("%s", str); 注意不能用 %c 也不需要取址符 &
      • printf("%s", str);
      • %s 识别空格作为字符串的结尾

    • getchar() 和 putchar()

      • 与上面所说的输入输出单个字符时的用法相同，注意吸收换行符

    • gets() 和 puts()

      • gets(str); 注意：gets()识别换行符作为输入结束，因此 scanf 完一个整数后，如果要使用 gets() ，请一定先用 getchar() 接收整数后的换行符
      • puts(str); 输出一串字符串，后面紧跟一个换行符

    • cin 和 cout

      • char str[100]; cin.getline(str, 100); 直接把一整行读入到 char 字符数组中，有效长度为输入长度-1，因为最后会默认在字符数组结尾加上 '\0'，如果输入字符串太长，会进行截断，识别回车符作为字符串结尾

      • string str; getline(cin, str); string 容器简便读入法，识别回车符为字符串结尾，也可以读取char[]

      • string str; cin >> str; 识别空格作为字符串结尾，空格后面的字符存放在输入流中，等待下次输入

      • string str; cout << str; 简便输出 string 类型字符串的方法

      • 用 printf 输出 string 的方法：需要转换为 char[] 类型

// c_str()函数的返回值是一个临时指针，所以需要用 strcpy()的方式转换为 char[]类型

注意：如果不是使用 scanf() 的 %s 或 gets() 输入字符串，例如使用 getchar() ，请一定要在输入的每个字符串最后加入 '\0'，否则 printf() 和 puts() 输出字符串会因无法识别字符串末尾而输出一大堆乱码。

可以吸收空格符的方法总结：gets(), cin.getline(char* s, len), getline(cin, string s)

遇到空格符即停止读取的方法：scanf("%s", char* s), cin >> string s 或 cin >> char* s

  • 字符数组常用函数 （cstring 头文件）（注意不适用于 string 容器）
    • int strlen(str) 返回结尾 '\0' 前的字符个数（简单来说就是字符串的长度）
    • int strcmp(str1, str2) 按字典序进行比较，返回值分为三种情况：
      • str1 > str2: 返回一个正整数（不一定是 1 ）
      • str1 < str2: 返回一个负整数（不一定是 -1 ）
      • str1 == str2: 返回 0
    • strcpy(str1, str2) 将 str2 字符数组复制给 str1 字符数组，包括结尾的结束符 '\0'
    • strcat(str1, str2) 将 str2 字符串拼接到 str1 字符串的后面
  • ** sscanf 和 sprintf **的用法：
    • sscanf(str, "%d", &n); 可以实现将 str 中的内容以 "%d" 的格式写到 n 中（字符数组-->整型）
    • sprintf(str, "%d", n); 可以实现将 n 以 "%d" 的格式写到 str 字符数组中（整型-->字符数组）
    • 其它类型与字符数组之间的转换也类似。

# 指针注意点

  • 指针是一个 unsigned 类型的整数
  • 连续定义指针时需要在每个变量前加上 *，例如： int* p1, p2, p3; 此时只有 p1 是 int* 类型，p2 和 p3 都是 int 类型，需要改为： int *p1, *p2, *p3;
  • 若 int* p = &a 那么修改 *p 也会修改 a
  • 数组名也可以作为数组第一个元素的指针，即 arr == &arr[0] 同时 arr + i == &arr[i]
  • 两个指针做减法，得到的结果不是它们指代的地址的值的差值，而是两个地址之间的距离（这个距离以指针基类型为单位），如 int* p1 = a; int* p2 = &a[5] 那么 p2 - p1 = 5 ≠ 20 意思为 p2 和 p1 之间相差了 5 个 int
  • 指针定义时一定要进行赋值，不然会指向位置的地址，造成出错
  • 指针作为参数传递时，在函数中直接改变指针指向的地址是没有用的（地址是个无符号整数，这个整数相当于按值传递），必须修改地址中存放的数据的值
  • 常量不可以使用引用，如果函数的形参是 &a 引用形式，那么调用时一定不能用常量作为实参

# 结构体注意点

  • 结构体声明的末尾要加上分号

  • 结构体声明末尾大括号后面可以直接定义变量：struct node {}s1, s2[10];

  • 结构体的构造函数

    • 语法：node(param1, param2, ...) {...} 或 node(_param1, _param2, ...): param1(_param1), param2(_param2), ..., {}

    • 和类(class)的构造函数一致，默认有个无参构造函数

    • 如果自己定义了构造函数，那么默认的无参构造函数就会没有，所以每次定义结构体变量都需要满足自己定义的构造函数

    • 当然为了灵活，可以定义多个不同类型的构造函数

# 关于浮点数精度的注意点

  • 浮点数之间的比较，通过设置一个很小的偏移量eps来比较（因为浮点数存储时不是完全精准的）
const double eps = 1e-8;
#define Equ(a, b) ((fabs((a) - (b))) < (eps))
#define More(a, b) ((a) - (b) > (eps))
#define Less(a, b) ((a) - (b) < (-eps))
#define MoreEqu(a, b) ((a) - (b) > (-eps))  // a 比 b 小在 -eps 以内就可以认为两者相等
#define LessEqu(a, b) ((a) - (b) < (eps))  // a 比 b 大在 eps 以内就可以认为两者相等

# 多点测试注意点

多点测试：大部分在线评测系统使用的评测手段，PAT 是单点测试

多点测试的输入应对类型：

  • while ... EOF 型：scanf 函数的返回值是其成功读入的参数的个数，读入失败时，scanf 函数会返回 -1 而不是 0，而 C 语言中使用 EOF (End Of File) 来代表 -1
    • 对于普通输入：**while (scanf("%d", &n) != EOF) **
    • 对于字符串：
      • while (scanf("%s", str) != EOF)
      • while (gets(str) != NULL)
  • while ... break 型：当输入为 0 时结束输入
    • 判断条件写为：while (scanf("%d%d", &a, &b), a || b) 只要有一个不为 0 就进入循环
  • while (T--) 型：题目给出测试数据的组数
    • 判断条件写为：while (T--)

多点测试的常见输出类型：

  • 正常输入，每两组输出数据中间没有额外的空行，即输出数据是连续的多行
  • 每组数据输出之后都额外一个空行
  • 两组输出数据之家有一个空行，最后一组数据后面没有空行：需要通过判断 T 是否已经减小到 0 来判断是否应当输出额外的空行

注意事项：

  • 在多点测试中，每一次循环都要重置一下变量和数组，否则在下一组数据来临的时候变量和数组的状态不是初始状态。

  • 重置数组一般使用 memset 函数或 fill 函数

# priority_queue 比较函数写法

写法 1：

struct node {
	int id, fre;
	node(int _id, int _fre): id(_id), fre(_fre) {}
	bool operator < (const node& no) const {
		return (fre == no.fre)? id < no.id: fre > no.fre;
	}
};
