# 1130 叶值的最小代价生成树（难度：中等  语言：C++）

# 1.题目

给你一个正整数数组 arr，考虑所有满足以下条件的二叉树：

  • 每个节点都有 0 个或是 2 个子节点。
  • 数组 arr 中的值与树的中序遍历中每个叶节点的值一一对应。（知识回顾：如果一个节点有 0 个子节点，那么该节点为叶节点。）
  • 每个非叶节点的值等于其左子树和右子树中叶节点的最大值的乘积。

在所有这样的二叉树中，返回每个非叶节点的值的最小可能总和。这个和的值是一个 32 位整数。

# 2.示例

输入：arr = [6,2,4]
输出：32
解释：
有两种可能的树，第一种的非叶节点的总和为 36，第二种非叶节点的总和为 32。

# 3.提示

  1. 2 <= arr.length <= 40
  2. 1 <= arr[i] <= 15
  3. 答案保证是一个 32 位带符号整数，即小于 2^31。

# 4.思路

方法一：单调递减栈 时间复杂度 空间复杂度

  • 数组 arr 中是按从左到右（中序遍历）存储的每片叶子。
  • 考虑两片连续的叶子 a 和 b, 则 a*b 是它们父结点的权值。在计算其它结点的权值时，由于是在左右两边各选出一片最大的叶子，又因为 a 和 b 在组合后，对于其他所有待算权值的结点来说肯定是在同一边的，即只会在 a 和 b 中选较大值参与计算，所以在组合完 a 和 b 后可以直接将 min(a, b) 抹去，只保留较大值参与之后的计算。
  • 而为了尽量保持所有结点权值之和小，每次相乘的两数也应该尽量小，并且树的高度也尽量小。也就是说，在数组 arr 中取出一个值 a ，其左边元素的值为 b，右边元素的值为 c，比较 b 和 c 的大小，选出较小的那个值与a相乘，并且在之后的计算中直接忽略a。
  • 因此可以每三个值 a b c 进行考虑，总共四种情况：
    1. a > b > c：直接将 b 和 c 相乘，丢弃较小值。
    2. a > b < c：将 b与 min(a, c) 相乘。
    3. a < b < c：将 a 与 b相乘。
    4. a < b > c：和 情况 2 一致。 感觉情况很多种，其实可以通过单调递减栈来统一。先在栈底加入 INT_MAX，作用是：当出现情况 3时，由于最左边是个大数，把 c 归入下一组，就变成 情况 2 了，出现 情况 4 时，把 c 归入下一组，也可以看成 情况 2。也就是说，把 2 3 4 三种情况的三个数，都归为 INT_MAX a b => 大 小 大 类型，当出现此类型时，也即下一个遇到的数比栈顶的数大时，进行 情况 2 的处理模式。
  • 采取策略：先创建一个栈，将 INT_MAX 加入栈底。之后遍历整个 arr 数组，每次检查遍历元素的值 t 与栈顶元素值 top 的大小，遇到比 top 小的值就入栈。如果遇到大值，因为栈顶现在是小值，之后计算不会考虑，就可以先将 top（由于最左边保证有 INT_MAX ，所以已经是 大 小 大 模式）pop 出栈，比较新栈顶的值与 t 的大小 (min(a, c))，将小值与 top 相乘。继续将栈顶值与 t 比较，重复操作，直至栈顶元素值大于 t，则将 t 入栈（保证单调递减栈）。最后遍历完 arr 后很有可能栈中还存留 size > 2 个递减的叶子值，则按后进先出原则两两相乘，并保留大值抛弃小值，直到栈中只剩下不超过两个元素（其中一个是 INT_MAX )。

# 5.题解

// 单调栈法
class Solution {
public:
    int mctFromLeafValues(vector<int>& arr) {
        stack<int>treeNode;
        int res = 0;
        int top;
        treeNode.push(INT_MAX);
        for (int t: arr) {
            while (treeNode.top() <= t) {
                top = treeNode.top();
                treeNode.pop();
                res += top * min(treeNode.top(), t);
            }
            treeNode.push(t);
        }
        while (treeNode.size() > 2) {
            top = treeNode.top();
            treeNode.pop();
            res += top * treeNode.top();
        }
        return res;
    }
};

// 动态规划 枚举法
class Solution {
public:
    int mctFromLeafValues(vector<int>& arr) {
        int n = arr.size();
        // dp[i][j]: arr 中区间[i, j-1]中的最小代价值
        int dp[n][n];
        // maxValueInInterval[i][j]: arr 中区间[i, j-1]中的最大叶子值
        int maxValueInInterval[n][n] = { 0 };

        // 枚举 arr 区间[i, j]的最大叶子值
        // dp[i][j]初始化为 INT_MAX, 因为之后用 min()挑选出最小的情况
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                int max = 0;
                for (int k = i; k <= j; k++) {
                    if (arr[k] > max) {
                        max = arr[k];
                    }
                }
                maxValueInInterval[i][j] = max;
                dp[i][j] = INT_MAX;
            }
        }

        // dp[i][i]为每个叶子结点，不算入结果，置 0
        for (int i = 0; i < n; i++) {
            dp[i][i] = 0;
        }

        // 枚举区间长度：1~n-1
        for (int length = 1; length <= n-1; length++) {
            // 枚举区间起始位置：0~n-length-1
            // 可以通过计算得到区间的终止位置：start+length
            for (int start = 0; start <= n-length-1; start++) {
                // 枚举区间中点的位置：start+1~start+length-1
                for (int mid = start; mid <= start+length-1; mid++) {
                    dp[start][start+length] = min(dp[start][start+length], 
                                                dp[start][mid] + dp[mid+1][start+length] 
                                                + maxValueInInterval[start][mid]
                                                * maxValueInInterval[mid+1][start+length]);
                }
            }
        }

        return dp[0][n-1];
    }
};
