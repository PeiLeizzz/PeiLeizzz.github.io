# 1021 删除最外层的括号（难度：简单  语言：C++）

# 1.题目

有效括号字符串为空 ("")、"(" + A + ")" 或 A + B，其中 A 和 B 都是有效的括号字符串，+ 代表字符串的连接。例如，""，"()"，"(())()" 和 "(()(()))" 都是有效的括号字符串。

如果有效字符串 S 非空，且不存在将其拆分为 S = A+B 的方法，我们称其为原语（primitive），其中 A 和 B 都是非空有效括号字符串。

给出一个非空有效字符串 S，考虑将其进行原语化分解，使得：S = P_1 + P_2 + ... + P_k，其中 P_i 是有效括号字符串原语。

对 S 进行原语化分解，删除分解中每个原语字符串的最外层括号，返回 S 。

# 2.示例

输入："(()())(())"
输出："()()()"
解释：
输入字符串为 "(()())(())"，原语化分解得到 "(()())" + "(())"，
删除每个部分中的最外层括号后得到 "()()" + "()" = "()()()"。

# 3.提示

  1. S.length <= 10000
  2. S[i] 为 "(" 或 ")"
  3. S 是一个有效括号字符串

# 4.思路

所谓原语，就是不能再拆分了，即所有的部分都容纳在最外层的 () 中了。

设置一个监测量 flag，用于监测(和)的数量，初始值设为 0。 设置字符串 str，作为 return 的结果。

  1. 每次遇到 (，flag++; 遇到 )，flag--，同时在适当时候进行字符串的拼接。
  2. 同时注意，输出时，每个原语最外层的()需要省略，也就是说，当 flag == 0 并且遇到 (，以及当 flag == 1 并且遇到 ) 时，都不能进行字符串的拼接。
  3. 这里把字符串拼接条件设为 flag > 0，此时如果遇到最外层的 )，flag 正好经过自减变为 0，所以剩下只需要考虑如何判断遇到每个原语最外层的 ( 即可。
  4. 这里我的办法是，当 flag == 0 并且遇到 ( 时，进行完 flag++ 后直接 continue 进行下一轮循环，这样正好可以规避拼接。

# 5.题解

class Solution {
public:
    string removeOuterParentheses(string S) {
        int flag = 0;
        string str = "";
        for (char c: S) {
            if (c == '(' && flag == 0) {
                flag++;
                continue;
            }
            else if (c == '(') {
                flag++;
            }
            else {
                flag--;
            }
            if (flag > 0) {
                str.push_back(c);
            }
        }
        return str;
    }
};
