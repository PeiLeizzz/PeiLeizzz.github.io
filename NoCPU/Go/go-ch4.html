# 复合数据类型

  • 数据（同构）和结构体（异构）是聚合类型，有固定的内存大小；slice、map 是动态的数据结构

  • 数组：

    • 初始化：默认隐式零值

      var q [3]int = [3]int{1, 2, 3} // [1, 2, 3]
var r [3]int = [3]int{1, 2} // [1, 2, 0]

      q := [...]int{1, 2, 3} // 自动计算长度

      q = [4]int{1, 2, 3, 4} // error

      数组太僵化，长度属于类型、不能添加或删除元素、传参复制，所以一般用 slice 替代数组（除非某些领域需要特定大小的数组）

  • Slice：变长同构序列，[]T，包含三部分：指针、长度、容量

    • 指针指向第一个 slice 元素对应的底层数据元素的地址，但 slice 的第一个元素并不一定就是数组的第一个元素。（slice 底层引用了数组）

    • len() 返回长度，slice 中元素的数目

    • cap() 返回容量，一般是从 slice 的开始位置到底层数据的结尾位置

    • 多个 slice 可以共享底层的数据，并且引用的部分区间可能重叠（修改其中一个 slice 可能会影响到另一个 slice）。切片 [st:en] 可以创建新的 slice

      months := [...]string{1: "January", /*...*/, 12: "December"}
Q2 = months[4: 7] // ["April" "May" "June"]
summer = months[6: 9] // ["June" "July" "August"]

      2
      3

    • 切片超过 cap(s) 会引发 panic 异常；超过 len(s) 会扩展 slice

      endlessSummer := summer[:5] // [June July August September October]
      1

      切片返回的的子序列共享底层数组；对于字符串或 []byte 切片则都是产生新的数据

    • 向函数传递 slice （一个指针）只是对底层的数组创建了一个新的 slice 别名（Go 语言只有值传递）

    • slice 初始化不用指定长度：会隐式的创建一个合适大小的数组

      s := []int{0, 1, 2, 3, 4, 5}
      1

      if summ == nil { /* ... */ }

    • make 函数创建 slice：make([]T, len) 或 make([]T, len, cap)，不指定 cap 则默认 == len

      在底层，make 函数创建了一个匿名的数组变量，然后返回一个 slice

    • append 函数追加元素：但不建议在函数参数 slice 在函数内部进行 append，因为函数内部会复制一个 slice 引用，如果扩容或者在 len 外添加元素（append），这个引用就不是原来的引用了，这样的话不会影响到原来的 slice（如果一定要在函数内部 append，记得返回新引用覆盖原引用）

      var runes []rune

  • 结构体：由零个或多个任意类型的值聚合成的实体，可以作为一个整体单元被复制
