# 入门

  • 运行：go run *.go

  • 编译：go build *.go

  • 自动修改依赖：go mod tidy

  • 包：package main // 表示该文件属于 main 包

  • 导入包：import "fmt"

  • main 包定义了一个独立可执行的程序，而不是一个库

  • 编译器会把特定符号后的换行符转换为分号，所以函数的左括号不能独占一行，x+y 只能在 + 后换行（以 + 结尾不会被插入分号）

  • 变量声明（会自动初始化）：var s string

  • 短变量声明：i := 1 根据变量的初始值为变量赋予适当的类型

    s := "" // 最简洁，但只能用在函数内部，不能用于包变量
var s string // 依赖默认初始化机制
var s = ""
var s string = ""

  • 创建 map make(map[string]int)，map 的迭代顺序是随机的

  • 读取标准输入并将其拆分为行或单词：input := bufio.NewScanner(os.Stdin)

  • 向标准错误流打印错误信息：fmt.Fprintf(os.Stderr, "dup2: %v\n", err)

  • 创建 channel ch := make(chan string)

  • 创建 goroutine go

  • 当一个goroutine尝试在一个channel上做send或者receive操作时，这个goroutine会阻塞在调用处，直到另一个goroutine从这个channel里接收或者写入值，这样两个goroutine才会继续执行channel操作之后的逻辑。

  • switch: 带操作对象，case 后跟操作对象的可能值；不带操作对象，case 后面跟表达式，与 true 比较（类似 if else）
