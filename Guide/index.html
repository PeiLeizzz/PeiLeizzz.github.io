# 博客内容

  • 博客的内容大多数都是学校里面课程的内容，算是平常学习的积累。以这样一种形式，也算让自己印象深刻，并且期末复习的时候有了存档也比较方便。
  • 大多数博文都是关于计算机编程方面的，可能会有少数基于个人兴趣的内容。但是相对而言，因为计算机方面的内容最适合以这种形式呈现。
  • 由于大部分都是学校中的内容，所以会比较简单，算法部分将是后期主要进行汇总的部分。另外，很多内容是从学校老师的课件中 COPY 过来的，也有很多书上的内容，如有侵权，请一定联系我，我会马上删除。

# 勘误信息

  • 由于这个博客是从原来一个地方刚刚迁移到这里，很多地方不一定能完全适配。比如 Markdown 和 的一些语法会有不同。迁移过来的时候也比较麻烦，但也大致检查修改了一下，如发现问题，欢迎提出。
  • 由于自己有强迫症，任何英文、数字与中文之间必须要有一个空格，如果您发现某处不满足，请提醒我，非常感谢。

# 博客搭建

  • 搭建静态博客需要一定的 git 基础知识，关于这部分内容，可以查看 Git教程。
  • 同时，此博客是基于 Vuepress 的 reco 主题搭建而成，如果您也想搭建一个像这样的博客，可以查看 Vuepress官方文档。
  • 或许您不喜欢这样的风格，那么可以试一下另一种静态网站 Hexo，关于这部分内容，可以参考 Hexo官方文档。
  • 撰写博文需要会使用 Markdown 及可能会用到的 语法，可以查看 Markdown教程 和 官方文档。
  • 推荐使用 VSCode 或者 Typora 来撰写博文。
