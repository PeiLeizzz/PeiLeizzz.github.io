# 非线性规划

  • 目标函数或者约束条件中至少有一个非线性函数，就是非线性规划问题()

  • 标准形式：

    \min f(x)\\\ s.t.\begin{cases} Ax\le B\\\ Aeq*x=Beq\\\ C(x)\le 0\\\ Ceq(x)=0 \end{cases}

  • MATLAB 函数：

[x,fval]=fmincon(fun,X0,A,b,Aeq,Beq,LB,UB,nonlcon,OPTIONS)
% A b Aeq Beq 定义线性约束
% fun：用M文件定义的函数f(x)
% nonlcon：用M文件定义的非线性向量函数C(x), Ceq(x)

# 二次规划

  • 非线性规划的目标函数为自变量 的二次函数，约束条件全是线性的，则为二次规划

  • 标准形式

  • MATLAB 函数：

[x,fval]=quadprog(H,f,A,b,Aeq,Beq,LB,UB,X0,OPTIONS)
% H：实对称矩阵（转置=本身）关于目标函数的二次项
% f：列向量 关于目标函数的一次项
