# 蛋白质在生命活动中的重要性

# 生命是蛋白体的存在方式

# 蛋白质是生命的物质基础，是生命活动的体现者和执行者

# 蛋白质是生物机体的结构物质

# 蛋白质是生物的功能物质

# 分类

# 根据分子的性状分类

  • 球状蛋白质

    • 组蛋白
    • 免疫球蛋白

  • 纤维状蛋白质

    • 指甲
    • 羽毛

# 根据组成分类

  • 单纯蛋白质

    • 组分只有阿尔法-氨基酸

  • 结合蛋白质

    • 由蛋白质和非蛋白质物质组成

      • 染色体蛋白
      • 血红蛋白
      • SOD

# 化学组成

# 元素组成

  • C H O N 以及少量的 S P Fe Zn Mn l

  • 各种蛋白的 N 含量都近于 16%

    • 测定蛋白

      • 定氮法

      • 凯氏定氮法

        • 蛋白质含量 = 无机氮含量 ✖️ 6.25
        • 1g N = 6.25g 蛋白质

# 组成单位

  • 氨基酸（AA）

    • 氨基酸是羧酸分子中阿尔法-碳原子上的一个氢原子被氨基替代而成的化合物，故称阿尔法-氨基酸

    • 特点

      • 构成蛋白质的氨基酸有 20 种（天然） 均为 阿尔法-氨基酸：L-(a)-AA

        • 脯氨酸除外

      • 阿尔法-氨基酸有两种构型

        • D构型

        • L构型

          • 蛋白质中的氨基酸都是L-型

      • 共同点为在同一碳原子上有羧基、氨基和氢 不同点是侧链 R 基不同

      • 两性电解质

        • 酸性：-COOH
        • 碱性：-NH2

      • 除甘氨酸（Gly，R=H）

        • 其他的阿尔法C原子都是手性碳原子，具有光学活性，也就有D和L两种异构体

    • 分类

      • 酸性氨基酸

        • R基团含羧基（酸性基团） pH = 7 时带负电

          • 谷氨酸（味精）
          • 天冬氨酸

      • 碱性氨基酸

        • pH = 7 时带正电

          • 赖氨酸（氨基）
          • 精氨酸（胍基）
          • 组氨酸（咪唑基）

      • 中性氨基酸

        • pH = 7 时不带电 15种

          • 非极性或疏水性氨基酸 8种
          • 极性不解离氨基酸 7种

      • 非编码氨基酸

        • 仅存在于少数蛋白质中 150多种

          • 鸟氨酸
          • 瓜氨酸
          • 胱氨酸

    • 必需氨基酸

      • 机体需要但机体不能合成或合成量少，不能满足需要，而要有食物供给的氨基酸

        • 上述20种氨基酸种有8种

          • Lys 赖氨酸
          • Trp 色氨酸
          • Val 缬氨酸
          • Phe 苯丙氨酸
          • Thr 苏氨酸
          • Leu 亮氨酸
          • Ile 异亮氨酸
          • Met 甲硫/蛋氨酸

    • 重要理化性质

      • 物理性质

        • 无色结晶

        • 熔点高 200-300度

        • 在水中溶解度不相同

        • 易溶于酸碱，一般不溶于有机溶剂

        • 旋光性

        • 光吸收

          • 20中氨基酸在可见光区都没有光吸收

          • 在远紫外区（<220nm）均有光吸收

          • 在近紫外区（220-300nm），只有酪氨酸、苯丙氨酸和色氨酸有吸收光的能力

            • 酪氨酸 275nm
            • 苯丙氨酸 257nm
            • 色氨酸 280nm

      • 化学性质

        • 氨基参与

          • 亚硝酸

          • 甲醛

          • 2，4-二硝基氟苯

            • 蛋白质一级结构氨基末端测定

          • 异硫氰酸苯酯

          • 酰化试剂

            • 丹磺酰氯

          • 形成西佛碱反应

          • 荧光胺

            • 测定氨基酸含量

          • 脱氨基反应

        • 羧基参与

          • 成酯成盐反应
          • 成酰胺反应
          • 脱羧反应
          • 叠氮反应

        • 氨基和羧基共同参与

          • 两性离解和等电点

      • 两性电离性质——最重要性质

        • 氨基酸所带静电荷为零时，溶液的pH值称为该氨基酸的等电点，以pl表示

        • pH = pl，两性离子形式存在

        • pH < pl，以正离子形式存在

        • pH > pl，以负离子形式存在

        • 特点

          • 在pl时，氨基酸的溶解度最小（不解离）

        • 应用

          • 通过调节氨基酸溶液的pH值，使不同的氨基酸在各自的等电点结晶析出，来分离提纯不同的氨基酸

        • 范围

          • 酸性氨基酸：pl 2.7～3.2

            • 2种

          • 中性氨基酸：pl 5.0～6.0

            • 15种

          • 碱性氨基酸：pl 9.5～10.7

            • 3种

    • 生理功能

      • 作为寡肽和多肽的组成单位

      • 作为多种生物活性物质前体

        • NO前体：精氨酸
        • 组胺前体：组氨酸
        • 褪黑素前体：色氨酸

      • 神经递质

      • 氧化分解成ATP

      • 糖异生前体

    • 氨基酸及其衍生物在医药中的应用

      • 氨基酸的营养

        • 必需氨基酸

        • 非必需氨基酸

          • 体内自身能从糖代谢产物合成

      • 氨基酸输液

        • 肝病

          • 此类输液要求富含支链氨基酸

        • 肾病

          • 补氮

      • 治疗消化道疾病

        • 保护消化道黏膜或促进黏膜增生

          • 谷氨酸
          • 谷氨酰胺

        • 调节胃酸浓度

          • 甘氨酸及其铝盐
          • 谷氨酸盐酸盐

      • 治疗脑及神经系统疾病

        • 谷氨酸钙盐和镁盐

          • 神经衰弱
          • 脑外伤
          • 癫痫

        • y-酪氨酸

          • 记忆障碍
          • 语言障碍
          • 脑外伤后遗症

        • L-色氨酸

          • 精神分裂症
          • 酒精中毒
          • 抑郁症

        • 5-羟色氨酸

          • 内因性抑郁症
          • 失眠
          • 偏头痛

      • 治疗肿瘤

        • S-氨甲酰-L-半胱氨酸

      • 赖氨酸

        • 碱性非必需氨基酸

        • 第一限制性氨基酸

        • 应用

          • 调节人体代谢平衡
          • 减肥
          • 治疗铅中毒

        • 婴儿需求量大

      • 色氨酸

        • 可以转化成5-羟色胺

        • 应用

          • 抗闷剂
          • 抗痉挛剂
          • 胃分泌调节剂
          • 胃黏膜保护剂
          • 强抗昏迷剂

      • 缬氨酸、亮氨酸、异亮氨酸、苏氨酸

        • 均属支链氨基酸

        • 应用

          • 缬氨酸

            • 治疗肝功能衰竭
            • 加快创伤愈合

          • 亮氨酸

            • 突发性高血糖症
            • 头晕治疗剂
            • 营养滋补剂

          • 异亮氨酸

            • 神经障碍
            • 食欲减退
            • 贫血
            • 肌肉蛋白质代谢

          • 苏氨酸

            • 脂肪代谢

      • 天冬氨酸、天冬酰胺

        • 使血液中的氨转变为尿素排泄出去
        • 合成乳清酸等核酸前体物质的原料
        • 天冬酰胺的类似物S-氨甲酰基-半胱氨酸可以对抗白血病

      • 胱氨酸、半胱氨酸

        • 含硫的非必需氨基酸

        • 降低人体对蛋氨酸的需要量

        • 胱氨酸形成皮肤

        • 半胱氨酸缓解中毒、防治放射线

        • 半胱氨酸衍生物

          • N-乙酰-L-半胱氨酸

            • 防治支气管炎

          • L-半胱氨酸甲酯盐酸盐

            • 治疗支气管炎
            • 鼻黏膜渗出性发炎

        • 半胱氨酸促进毛发生长，治疗秃发症

      • 甘氨酸

        • 参与嘌呤类等合成

        • 甘氨酸+谷氨酸+丙氨酸

          • 防治前列腺肥大并发症
          • 排尿障碍
          • 频尿
          • 残尿

      • 组氨酸

        • 对幼儿是必需氨基酸
        • 治疗尿毒症
        • 防治贫血
        • 治疗消化道疾病，扩张血管 用于心绞痛、心功能不全
        • 类风湿性关节炎
        • 过敏性疾病如哮喘

      • 谷氨酸

        • 兴奋性递质，哺乳动物中枢神经系统中含量最高的氨基酸，兴奋作用仅限于中枢

        • 脱羧形成y-氨基丁酸

          • 抑制中枢神经兴奋
          • 记忆障碍、言语障碍、麻痹、高血压

        • y-氨基beita-羟基丁酸

          • 局部麻醉、记忆障碍、言语障碍、本能性肾性高血压、羊癫疯、精神发育迟缓

        • 治疗肝昏迷

        • 胃溃疡

      • 牛磺酸

        • 普遍存在于动物乳汁、脑、心脏、肌肉中含量最高
        • 游离形式存在，不参与蛋白质代谢
        • 体内牛磺酸是由半胱氨酸代谢而来
        • 缺乏会影响生长、视力、心脏、脑正常生长

# 营养价值

# 蛋白质的生理价值：食物中蛋白质在生物机体内的利用率

  • = 氮的保留量 / 氮的吸收量 * 100%

  • 氮的保留量

    • 用于机体生长或修补细胞及组织所消耗的蛋白质的量

  • 氮的吸收量

    • = 食入量 - 粪便排出量

  • 氮总平衡（氮平衡）

    • 机体摄入的氮量和机体排除的氮量相等

  • 氮负平衡

    • 食入氮量 < 排出氮量

      • 营养不良
      • 消耗性疾病

  • 氮正平衡

    • 食入氮量 > 排出氮量

      • 生长发育
      • 疾病恢复

  • 食物中蛋白质的营养价值取决于其 AA 组成

# 蛋白质的互补作用

  • 所谓蛋白质的互补作用就是指将必需 AA 缺乏或不足而又不相同的两种或几种蛋白适当配合于膳食中同时食用，以收到取长补短、互通有无的效果，使单独使用时无生理价值的具有一定的生理价值，生理价值低的有所提高。

# 结构

# 空间结构（高级结构）

  • 构型

    • 指光学活性不对称的一种特定的立体结构，对于含某一不对称碳原子的分子而言，是该不对称碳原子上各取代集团或原子在空间的排列
    • 构型的改变一定要有共价键的变化

  • 构象

    • 又称三维结构，立体结构，空间结构或高级结构等。是指分子中各原子、各集团之间的相互立体关系。从理论上说，构象可以有无数种，因为一个构象的改变是由于【单键的旋转】而造成的，不需要共价键的变化

  • 维持蛋白质空间构象的作用力

    • 氢键：a-螺旋，beita-折叠
    • 疏水作用：形成球蛋白的核心
    • 范德华力：稳定紧密堆积的基团和原子
    • 离子键：稳定a-螺旋，三、四级结构
    • 二硫键：稳定三、四级结构
    • 配位键：与金属离子的结合

  • 蛋白质结构的层次

    • 一级结构：氨基酸序列

    • 二级结构：a螺旋，beita折叠

    • 超二级结构

      • aa
      • beitabeita
      • beitabeitabeita
      • beita-a-beita

    • 结构域

    • 三级结构：所有原子空间位置

    • 四级结构：蛋白质多聚体

# 一级结构

  • 定义：即蛋白质的化学结构，指蛋白质分子中多肽链AA种类、数量和排列顺序及它们之间连接键的性质。 它是蛋白质作用的特异性、空间结构的差异性和生物学功能多样性的基础。

  • 成肽作用

    • 一个AA分子的a-COOH与另一分子AA的a-NH2脱水缩合而形成化合物的作用

    • 肽键

      • 连接这两个AA的化学键

    • 由N个AA组成的肽，就有N-1个肽键，称N肽

  • 多肽链结构

    • N末端向C末端合成
    • AA残基：多肽链中AA，由于参与肽键的形成，已非原来完整的分子
    • 肽链之间二硫键

  • 常见多肽

    • 谷胱甘肽：Glu-Lys-Gly（谷氨酰胱氨酰甘氨酸）
    • 催产素，加压素，血管舒缓素——9肽
    • 促肾上腺皮质激素（ACTH）——29肽

  • 肽与蛋白质的区别

    • 蛋白质：MW>6000(10000) 有自己特征的三维结构
    • 多肽：MW<6000(10000) 不一定有三维结构，有时有但不稳定

# 二级结构

  • 蛋白质的二级结构：指多肽链沿一定的轴盘旋或折叠所形成的一个有规则的构象，如a-螺旋，beita-折叠

  • 多肽链的主链在一级结构的基础上以肽键平面为基本单位的肽链通过氢键形成而折叠、盘曲形成的主链构象，它只涉及肽链主链的构象及链内或链间形成的氢键

    • 二级结构是由相邻的氨基酸残基的侧链基团决定的

  • 氢键是主要维持键：一个N原子H与另一个C原子O所形成的键

  • 肽平面

    • 肽键：0.132 nm

  • 类型

    • a-螺旋

      • 定义：多肽链的主链原子沿一中心轴盘旋所形成的右手螺旋结构

      • 是最常见、最典型、含量最丰富的二级结构

      • 右手比左手稳定，不是对映体

      • 左手是手性结构，具有旋光性

      • 特点

        • 以肽键平面为基本单位，Ca为转折，绕中心轴形成右手螺旋
        • 棒状结构，紧密卷曲的多肽主链形成棒的内部，侧链向外伸展
        • 每圈螺旋占3.6个氨基酸残基，沿螺旋轴方向上升0.54nm（螺距）。每个残基绕轴旋转100度，沿轴上升0.15nm
        • 氨基酸残基侧链‘R’伸向外侧
        • 相邻螺圈间形成链内氢键，氢键去向几乎与中心轴平行
        • 从N-末端出发，氢键是由每个肽基的C=O与其前面第5个肽基的N-H之间形成的

      • 影响因素

        • 与氨基酸的组成和排列顺序有关，所带电荷同性相斥（影响氢键生成）不利于生成
        • 较大的R侧链基团，不利于生成
        • 甘氨酸残基，二面角难形成，所以a-螺旋难形成
        • 脯氨酸残基，不能形成链内氢键

    • beita-折叠

      • beita-折叠又称beita-片层，指两条或多条几乎完全伸展的多肽链侧向平行或反平行排列聚集在一起，由相邻肽链主链上的-NH和C=O之间形成有规则的氢键维持的多肽构象；或一条肽链内的不同肽段间靠氢键而形成的构象

      • 特点

        • 肽链几乎完全伸展，所有肽键都参与链间氢键的交联，氢键与肽链的长轴接近垂直
        • 多肽主链取锯齿状折叠构象
        • 纤维蛋白中主要是反平行；球蛋白中反平行和平行均广泛存在。反平行折叠片中重复周期为0.7nm，平行中为0.65nm

      • 影响因素

        • R集团过大，并带同种电荷
        • 是纤维状蛋白质的基本构象，也存在于球状蛋白质中

    • beita-转角结构

      • 蛋白质分子多肽链上经常出现180度的回折，在这种肽链的回折角上就是beita-折角结构，它是由第一个氨基酸残基的C=O与第四个氨基酸残基的N-H之间形成氢键（发夹结构）

      • 富含Ala，Pro

      • 特点

        • 都有四个氨基酸残基，弯角处第一个残基的C=O和第四个残基的N-H之间形成氢键，产生一种不稳定的环形结构
        • 存在于多肽链的转角部位，多存在于球状蛋白质中

    • 无规则卷曲

      • 又称自由回转、自由绕曲。是指没有一定规律的松散肽链结构
      • 酶的功能部位常常处于这种构象区域里

# 超二级结构

  • 是指若干相邻的二级结构中的构象单元彼此相互作用，形成有规则的，在空间上能辨认的二级结构组合体。通常有beita-a-beita，beitabeitabeita，aa，beitabeita

# 结构域

  • 是指在超二级结构基础上进一步卷曲折叠成紧密的近似球状的结构，在空间上彼此分隔，各自具有部分生物功能的结构

# 三级结构

  • 是指多肽链在二级结构的基础上，由氨基酸残基侧链的相互作用使多肽链进一步盘旋和折叠，导致整个分子形成很不规则的特定构象

# 四级结构

  • 是指由两个或两个亚基（具有三级结构的多肽链）之间相互作用聚合而成的更复杂的构象

  • 亚基

    • 也称亚单位，原聚体，单体。亚基一般由一条多肽链组成，也可由两条或两条以上的多肽链构成，由亚基相互作用构成具有四级结构的的蛋白质称为寡聚肽（寡聚蛋白）

# 结构与功能

  • 一级结构

    • 一级结构不同，生物特性各异

      • 加压素与催产素

    • 一级结构中的关键部分相同，其他部分可以不同，但功能相同

    • 活性中心

      • 活性中心是指由几个氨基酸组成的一个空间区域。蛋白质的生物活性集中表现在这一空间区域，并与这一区域中的氨基酸组成，排布等密切相关

    • 一级结构中的关键部分（活性中心）变化，使生物活性改变或者丧失，甚至造成遗传性疾病

      • 镰刀形贫血症

  • 二级结构

    • 酶原的激活

      • 体内多数酶以物活性的酶原的形式存在。当机体需要时再转变成有活性的酶，这称为酶原的激活
      • 激活过程的本质是通过去除部分肽链片段，引起空间结构的变化而形成特定的空间构象，生成或暴露催化作用所必需的活性中心，这样的酶才表现为生物活性

    • 蛋白质前体的活化

      • 许多蛋白质（如蛋白质激素）在体内往往以前体的形式贮存，它们无活性或活性很低。
      • 胰岛素原经专一性酶的作用，切掉多余的AA，最终表现为完整生物活性

# 分子的大小

  • 相对分子量：10000～1000000

  • 测定分子量的主要方法

    • 分子筛层析法

      • 常用颗粒：葡聚糖，树脂
      • 凝胶颗粒为细微的多孔网状结构
      • 原理是分子筛效应（分子筛层析）

      • 利用人工合成的凝胶作为支持介质的电泳

      • 一定范围内，蛋白质（亚基）的迁移率只与其分子量大小的对数呈线性关系 LogM = K-a✖️de/d0

    • 生物质谱

# 变性

  • 定义：在某些理化因素的作用下，蛋白质严格的空间结构（不包括肽键）被破坏，从而引起若干理化性质改变和生物学功能丧失的现象

  • 因素

    • 物理

      • 高温、高压
      • 紫外线、X射线
      • 电离辐射和超声波

    • 化学

      • 有机碱、生物碱
      • 有机溶剂、重金属盐
      • 高浓度尿素、盐酸胍

  • 实质

    • 破坏了空间结构，一级结构不受影响（分子组成，MW不变）

  • 特点

    • 生物学活性丧失

    • 理化性质改变

      • 溶解度下降，沉降率提高
      • 粘度提高
      • 扩散速度下降
      • 旋光性、光吸收改变

    • 易被蛋白酶水解，空间结构改变

  • 分类

    • 可逆变性

      • 蛋白质变性后如将变性剂除去，该蛋白质分子的天然构象和生物学活性还能恢复——复性

        • 本质

          • 一级结构决定高级结构

        • 为什么能复性？

          • 在一定的环境条件下，蛋白质的一级结构能够决定二、三、四级结构。变性蛋白质的二、三、四级结构虽然遭到破坏，但是一级结构仍然保持不变，因此除去变性剂后，在适当的环境条件下，蛋白质能卷曲折叠成为【能量最低的构象】，一般天然蛋白质正是具有这种能量最低的构象

    • 不可逆变性

      • 若变性条件强烈，作用时间长，构象变化大，理化性质难以恢复

  • 常见变性剂

    • 尿素
    • 盐酸胍
    • 硫基乙醇
    • SDS 十二烷基硫酸钠

# 两性性质及等电点

  • 等电点

    • 在某一pH值溶液中，Pr酸性基团和碱性基团的解离程度相当，Pr分子所带正负电荷相等，净电荷为零，在电场中既不向阳极也不向阴极移动，此时溶液的pH值称为该蛋白质的等电点 （净电荷为零的蛋白质溶液的pH值）

    • 是鉴别蛋白质的指标之一

    • 与蛋白质分子中所含的氨基酸种类有关

    • pH = pl 净电荷为零，在电场中不移动

      • 溶解度最小，易沉淀

    • pH > pl 净电荷为负，向阳极移动

    • pH < pl 净电荷为正，向阴极移动

# 胶体性质

  • 利用蛋白质不能透过半透膜的性质，可以对蛋白质进行纯化

  • 溶液能形成稳定胶体的因素

    • 极性不电离基团：形成水化层
    • 极性电离基团：形成双电子层

# 沉淀反应

  • 中性盐沉淀

    • 低浓度：盐溶作用

      • 低盐浓度使蛋白质表面吸附某种离子，与水分子作用增加

    • 高浓度：盐析作用

      • 破坏水化层中和电荷

  • 有机溶剂沉淀

    • 乙醇、丙酮、甲醇
    • 使蛋白质表面失去水化层，相互凝聚而沉淀
    • 变性，不可逆 除非短时，低温

  • 加热沉淀

    • 在等电点最易沉淀
    • 变性，不可逆

  • 重金属盐沉淀

    • 蛋白质带负电基团可与重金属离子结合成沉淀
    • 变性，不可逆
    • 抢救误食重金属盐中毒的病人

  • 生物碱试剂沉淀

    • pH < pl 时，蛋白质带正电，可与一些生物碱试剂结合成沉淀
    • 变性，不可逆

  • 双缩脲

# 免疫学反应

  • 能诱导机体形成抗体的大分子外来异物称为抗原

  • 性质

    • 异原性
    • 大分子性
    • 特异性

  • 抗原的特异性决定于抗原分子表面的特殊化学基团，称为抗原决定簇

  • 抗体

    • 动物针对外来物质的出现而合成的一种蛋白质，它是由相应抗原刺激机体免疫系统所产生的免疫球蛋白

  • 抗原抗体反应

    • 特异性结合

    • 原理

      • 抗原决定簇和抗体分子超变区的相互作用，是一种分子表面的特异性的可逆的弱结合力

    • 极短距离内，空间结构必须吻合，具有互补性

    • 特点

      • 特异性

      • 比例性

        • 通常把最迅速出现沉淀时的抗原抗体浓度比或量比称为抗原抗体反应的最适比

      • 可逆性

        • 非共价键结合

      • 敏感性

  • 单克隆抗体

    • 单克隆抗体是由单一B细胞产生的高度均一、仅针对某一特定抗原表位的抗体

    • 优点

      • 特异性强，效价高
      • 产量大

    • 意义

      • 疾病诊断
      • 靶向给药

# 分离与纯化

  • 根据溶解度不同

    • 溶液的pH—等电点沉淀
    • 离子强度—盐析沉淀
    • 溶剂性质—低温有机溶剂沉淀

  • 根据分子大小不同

    • 透析

      • 利用蛋白质大分子对半透膜的不可透过性而与其他小分子物质分开

    • 超滤

      • 根据分子大小和形状，在10^-8cm数量级进行选择性分离的技术
      • 原理是利用超滤膜在一定的压力或离心力的作用下，大分子物质被截留而小分子物质则滤过排出
      • 超滤膜，是一种孔径规格一致，额定孔径范围为0.001-0.02微米的微孔过滤膜。在膜的一侧施以适当压力，就能筛出小于孔径的溶质分子，以分离分子量大于500道尔顿、粒径大于2～20纳米的颗粒。

    • 分子排阻层析

      • 葡聚糖凝胶

        • G（葡聚糖与交联剂的比例）越小、交联度越大、孔径越小。

      • 丙烯酰胺凝胶

      • 琼脂糖凝胶

    • 密度梯度离心

      • 蛋白质颗粒的沉降速度取决于它的大小和密度
      • 质量和密度大的颗粒沉降得快

  • 根据电离性质不同

    • 电泳法

      • 带电质点在电场中向电荷相反的方向移动，这种性质，称为电泳

      • 蛋白质在电场中移动的速度和方向主要取决于蛋白质分子所带的电荷的性质、数量及质点的大小和形状。

      • 外界影响因素

        • 电场强度
        • 溶液的pH
        • 离子强度
        • 电渗

      • 类型

        • 醋酸纤维薄膜电泳

        • 聚丙烯酰胺凝胶电泳

        • 等电聚焦电泳

          • 两性电解质作为支持物，电泳时即形成一个由正到负极逐渐增加depH梯度

          • 分辨率高，pl相差0.02pH单位即可分开

          • 对两性电解质的要求

            • 缓冲能力强
            • 有良好的导电性
            • 分子量小
            • 不干扰被分析的样品

        • 免疫电泳

          • 电泳技术和抗原与抗体反应的特异性结合

        • 二维电泳

          • 目前分辨率最高，信息量最多的技术

    • 离子交换层析

      • 利用离子交换基质上的可解离活性基团对各种蛋白结合力不同而达到分离目的的一种分离方法
      • 蛋白质是两性物质，所以既可以用阳离子也可以用阴离子
      • 洗脱方法：改变盐浓度和改变pH的方法

  • 根据配基特异性

    • 亲和层析

      • 利用生物分子与配基之间所具有的专一而又可逆的亲和力，使生物分子分离纯化的技术
      • 专一性强，理论上可以从复杂的体系中一步提取所需的物质。
