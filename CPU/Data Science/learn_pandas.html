    • 运算操作

    # Series 数据类型

    Series 类型由一组数据及与之相关的数据索引组成——一维的、带标签的数组

    创建：

    • Python 列表、字典
    • 标量值
    • ndarray
    • 其他函数，如 range()

    基本操作：包括 index 和 values 两部分，既类似 ndarray，也类型 dict

    • 索引 []
    • 可使用 NumPy 中的运算和操作
    • 通过自定义索引的列表进行切片
    • 通过自动索引进行切片
    • 可以使用 in(只判断自定义索引)，.get()
    • Series 对象和索引都可以有一个名字，存储在属性 .name 中
    • 修改：直接赋值

    当索引只生成一个值时，则只输出对应的值；反之，生成的还是 Series 类型

    # DataFrame 类型

    DataFrame 类型由共用相同索引的一组列组成，每列值类型可以不同——表格

    行索引和列索引：

    • 纵向：index axis=0
    • 横向：column axis=1

    创建：

    • 二维 ndarray 对象
    • 由一维 ndarray、列表、字典、元组或 Series 构成的字典
    • Series 类型
    • 其他的 DataFrame 类型

    切片：

    • 列：[column_name] Series 类型
    • 行：.loc[column_name] Series 类型
    • [[]] 切片则为 DataFrame 类型

    # 数据类型操作

    • 增加和重排：重新索引

      .reindex(index=None, columns=None, ...)
      1
      • index, columns 行列索引
      • fill_value 重新索引中，用于填充缺失位置的值
      • method 填充方法，ffill 当前值向前填充，bfill 向后填充
      • limit 最大填充量
      • copy 默认 True，生成新的对象，False 时，新旧相等不复制

    • 对索引类型（Index）的操作，

      索引类型不可改变，都是生成新的 Index 对象

      • .append()
      • .diff(idx) 计算差集
      • .intersection(idx) 交集
      • .union(idx) 并集
      • .delete(loc)
      • .insert(loc, e)

    • 删除：drop()

    # 数据类型运算

    算术运算法则：

    • 根据行列索引，补齐后运算，运算默认产生浮点数
    • 补齐时缺项填充 NaN
    • 二维和一维、一维和零维间为广播运算
    • 二元运算产生新的对象
      • + - * /
      • .add() .sub(). mul() .div() 可以增加额外参数

    比较运算法则：

    • 只能比较相同索引的元素，不进行补齐，同维尺寸必须相同
    • 二维和一维、一维和零位间为广播运算
    • 采用 > < >= <= == != 等符号进行的二元运算产生布尔对象

    # 数据的排序

    摘要：有损地提取数据特征的过程

    • 基本统计（含排序）
    • 分布/累计统计
    • 数据特征：相关性、周期性等
    • 数据挖掘（形成知识）

    数据排序：

    • .sort_index(axis=0, ascending=True)：在指定轴上根据索引进行排序，默认升序

    • .sort_values(axis=0, ascending=True)
      1

      :在指定轴上根据

      数值

      进行排序，默认升序

      • Series: .sort_values(axis=0, ascending=True)

    # 数据的相关分析

    相关性：

    • 正相关：X 增大，Y 增大
    • 负相关：X 增大，Y 减小
    • 不相关：X 增大，Y 不变

    协方差：

    公式：

    • 协方差 > 0：正相关
    • 协方差 < 0：负相关
    • 协方差 = 0：独立无关

    Pearson 相关系数:

    公式：

    • 0.8-1.0 极强相关
    • 0.6-0.8 强相关
    • 0.4-0.6 中等程度相关
    • 0.2-0.4 弱相关
    • 0-0.2 极弱相关或不相关

    相关分析函数：

    • .cov() 计算协方差矩阵
    • .corr() 计算相关系数矩阵，Person、Spearman、Kendall 等系数
